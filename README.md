# Focus Mode Website Blocker

A simple Python productivity tool that blocks distracting websites like Instagram and YouTube by modifying the system hosts file.

## Features
- Block distracting websites
- Unblock them anytime
- Helps improve focus while studying

## Tech Stack
Python

## How to Run

1. Run the script:
python blocker.py

2. Choose:
1 -> Block websites  
2 -> Unblock websites
